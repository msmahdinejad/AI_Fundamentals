# Phase 1: Search, Regression, and Local Search

This phase contains three AI fundamentals tasks:

- Grid search for an Angry Birds-inspired environment using BFS, UCS, iterative deepening/depth-limited search, and A*.
- Linear regression for asteroid diameter prediction with preprocessing, evaluation metrics, and SGD implemented from scratch.
- DNA center finding with local search, hill climbing, simulated annealing, and brute-force comparison.

Run the search scripts from `search_algorithms/` so the provided `env/` package and assets resolve correctly.
